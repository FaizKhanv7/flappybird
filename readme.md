# Flappy Bird (But Reversed)
This is basically flappy bird EXCEPT you are controlling the PIPES. So instead of trying to go through the pipes you have to make way for the bird :)

# How to play
This project is completely open source so you can download the code from this repo and open the `index.html` file in any browser to play locally and make changes OR you could play the deployed version with no setup required @ https://linktogame.com
